# Yash Kalsariya Portfolio

Premium personal portfolio built with Next.js App Router, TypeScript, Tailwind CSS, shadcn-style components, Framer Motion, Lenis, React Hook Form, Zod, Sonner, next-themes, Lucide Icons, and React Icons.

## Content Source

The content is primarily extracted from `Profile.pdf`:

- Name: Yash Kalsariya
- Title: Full Stack Developer
- Location: Ahmedabad, Gujarat, India
- Email: yashkalsariya040@gmail.com
- Experience: ThitaInfo, Sonoma Infotech Pvt. Ltd., CloudThat
- Education: BE Computer Engineering, SAL Institute of Technology and Engineering Research / Gujarat Technological University

Fields missing from the resume, such as phone number, detailed project links, and some certifications, are marked with professional placeholder content in `data/portfolio.ts`.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Useful Files

- `data/portfolio.ts` - edit profile content, skills, experience, projects, education, certifications, achievements, and social links
- `components/` - reusable site sections and interactions
- `components/ui/` - shadcn-style primitives
- `app/globals.css` - theme tokens, global styles, gradients, and motion safety
- `public/Profile.pdf` - downloadable resume
- `public/profile-placeholder.svg` - profile placeholder asset
